# Minimal Journal

A **privacy-first, offline-only personal journal**.  
Designed to be ultra-minimal, lightweight, and device-based.

## Core Principles
- No accounts, no tracking, no analytics
- Data stays only on your device (localStorage)
- Works completely offline
- Dark mode default, toggleable

## Features
- Write new entries with **CAPS toggle**  
- Mark entries as **IMPORTANT**
- **Reflection mode**: read, copy, delete, toggle important  
- **Filter**: show only important entries  
- **Calendar**: Monday-first, shows days with entries, click to filter by day  
- Ctrl+Enter or SAVE button to add entries  
- Minimal, distraction-free interface

## Data Control
- Copy entries as plain text  
- Backup via browser storage (localStorage)  
- Restore entries by pasting saved data  

## Delete Confirmation
- Deleting entries asks for user confirmation

## Hosting
- Works as a **single `index.html` file**  
- Perfect for GitHub Pages or local device use

## Privacy
- No automatic sync  
- No network calls  
- Your journal is **yours alone**
